Distribution Structure of the Qualitas Corpus

This describes the structure of the distribution form of the corpus. See also the structure of the corpus itself.

A distribution is organised as follows, where YYYYMMDD indicates the release of the corpus, and d (optional) indicates a specific distribution of the release. 

QualitasCorpus-YYYYMMDDd
  |
  +-- README
  |
  +-- Systems
  |
  +-- bin
  |    |
  |    +-- install.pl
  |
  +-- docs
  |
  +-- metadata
README
Fast startup details.
Systems
The contents of the corpus.
bin
Various executables (only scripts at the moment) potentially useful for using the corpus. In particular, install.pl is used for installation.
docs
The documentation for the corpus.
metadata
Metadata The metadata from all of the .properties files in the corpus but combined into a single file.
