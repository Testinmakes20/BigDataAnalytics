Installing the Qualitas Corpus

The installation process is slightly different, depending on whether the distribution is on writable or non-writable media. If the distribution is a tar or zip file, unpack it in the desired location of the corpus. Execute the perl script install.pl found in the bin directory (see the distribution structure). See the instructions for install.pl for more details.

The install.pl script unpacks the various archive files and does other organsiation, and is necessary in order for the corpus to be in a state useful for analysis.

Important Note

Due to licensing restrictions the JRE binaries and source is not in this distribution. In order to include the JRE in a given installation, it must be downloaded directly from Oracle and installed manually. Instructions are given in the relevant directory within Systems.
